**agents/analyst.py — implement run_analyst(state: AgentState) -> AgentState:**

Write tests first in tests/test_analyst.py (see Task 2), then implement to make them pass.

Logic:
1. Read thresholds from env: `alert_threshold = float(os.getenv("TONER_ALERT_THRESHOLD", "20"))`, `critical_threshold = float(os.getenv("TONER_CRITICAL_THRESHOLD", "10"))`
2. If `state["poll_result"] is None`, set `alert_needed=False`, append to decision_log, return early
3. Iterate `poll_result["readings"]` — for each `TonerReading`:
   - If `quality_flag == QualityFlag.BELOW_LOW_THRESHOLD.value`: append `{"color": color, "urgency": "CRITICAL", "display_value": "below low threshold (unquantified)"}` to flagged list. Do NOT check `data_quality_ok` first — BELOW_LOW_THRESHOLD is alert-worthy despite `data_quality_ok=False`.
   - Else if `data_quality_ok=True` and `toner_pct is not None`:
     - If `toner_pct < critical_threshold`: append with urgency="CRITICAL", display_value=f"{toner_pct}%"
     - Elif `toner_pct < alert_threshold`: append with urgency="WARNING", display_value=f"{toner_pct}%"
   - All other flags (UNKNOWN, SNMP_ERROR, NULL_VALUE, NOT_SUPPORTED, OUT_OF_RANGE): skip silently
4. Set `state["alert_needed"] = len(flagged) > 0`
5. Set `state["flagged_colors"] = flagged`
6. Append to decision_log: `f"analyst: {len(flagged)} colors flagged, alert_needed={state['alert_needed']}"`
7. Return state

Use `state["decision_log"] = state["decision_log"] + [entry]` (list concatenation, not .append()) to stay compatible with LangGraph's reducer in Phase 4.

Module-level logger: `logger = logging.getLogger(__name__)`
