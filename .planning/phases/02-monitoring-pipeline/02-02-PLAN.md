Implement guardrails/safety_logic.py with these public functions:

**`run_policy_guard(state: AgentState) -> AgentState`**
- If `state["alert_needed"]` is False, return state unchanged (no-op)
- Run `check_data_freshness(poll_result)` — if stale, call `log_suppression()`, set `state["suppression_reason"]`, set `state["alert_needed"]=False`, return
- Run `check_snmp_quality(poll_result)` — if snmp_error is set, suppress with reason="data_quality"
- Run `check_rate_limit(printer_host)` — if within 24h, call `log_suppression()`, suppress, return
- If all checks pass, return state unchanged (alert_needed still True, communicator proceeds)
- Append to decision_log for each check outcome

**`check_data_freshness(poll_result: PollResult) -> tuple[bool, str | None]`**
- `stale_minutes = float(os.getenv("STALE_THRESHOLD_MINUTES", "120"))`
- `poll_ts = datetime.fromisoformat(poll_result["timestamp"])`
- `now = datetime.now(timezone.utc)` — MUST use timezone.utc to avoid naive/aware TypeError
- `age_minutes = (now - poll_ts).total_seconds() / 60.0`
- If age > stale_minutes: return `(False, f"stale_data: age={age_minutes:.1f}min > threshold={stale_minutes}min")`
- Else: return `(True, None)`

**`check_snmp_quality(poll_result: PollResult) -> tuple[bool, str | None]`**
- If `poll_result["snmp_error"] is not None`: return `(False, f"data_quality: snmp_error={poll_result['snmp_error']}")`
- Else: return `(True, None)`

**`check_rate_limit(printer_host: str, state_path: Path = ALERT_STATE_PATH) -> tuple[bool, str | None]`**
- Load alert_state via `_load_alert_state(state_path)`
- If `last_alert_timestamp` for this host exists and `(now - last_ts) < timedelta(hours=24)`: return `(False, f"rate_limit: last_alert={last_ts_str}")`
- Else: return `(True, None)`

**`record_alert_sent(printer_host: str, state_path: Path = ALERT_STATE_PATH) -> None`**
- Read-modify-write: load state, set `state[printer_host]["last_alert_timestamp"] = now.isoformat()`, save

**`log_suppression(printer_host: str, reason: str, extra: dict, log_path: Path = persistence.LOG_PATH) -> None`**
- Build record: `{"event_type": "suppressed_alert", "printer_host": printer_host, "timestamp": now.isoformat(), "reason": reason, **extra}`
- Call `append_poll_result(record, log_path=log_path)`

**`_load_alert_state(state_path: Path) -> dict`**
- If not exists: return `{}`
- Try `json.loads(state_path.read_text(encoding="utf-8"))`, except `(json.JSONDecodeError, OSError)`: return `{}`

**`_save_alert_state(state: dict, state_path: Path) -> None`**
- `state_path.parent.mkdir(parents=True, exist_ok=True)`
- `state_path.write_text(json.dumps(state, indent=2), encoding="utf-8")`
