Create adapters/persistence.py. This module provides two functions: one to append a PollResult to the JSONL log and one to read the full history back.

Implement:

1. LOG_PATH = Path("logs/printer_history.jsonl")
   (Use Path from pathlib; the path is relative to the project root where main.py runs)

2. append_poll_result(result: PollResult, log_path: Path = LOG_PATH) -> None
   - Create the parent directory with log_path.parent.mkdir(parents=True, exist_ok=True)
   - Open with jsonlines.open(str(log_path), mode="a") — ALWAYS mode="a", never "w"
   - Write the result dict with writer.write(result)
   - Close via context manager (with statement)
   Comment in code: "mode='a' is critical — 'w' would truncate the entire history on every poll"

3. read_poll_history(log_path: Path = LOG_PATH) -> list[PollResult]
   - If log_path does not exist, return []
   - Open with jsonlines.open(str(log_path), mode="r")
   - Return list(reader)

The log_path parameter with a default allows tests to inject a temp path without patching globals.

Import from state_types: PollResult (for type annotation only — TypedDict at runtime is just a dict, no runtime enforcement needed).

CRITICAL: QualityFlag is a str Enum so its .value is already a plain string. When writing to the log, quality_flag fields will already be plain strings — no special serialization step needed.
