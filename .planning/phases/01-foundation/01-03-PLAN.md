Implement adapters/smtp_adapter.py. The SMTPAdapter wraps smtplib to send alert emails via STARTTLS. It must support mock mode for development and a real Outlook account for production.

Structure:

1. SMTPAdapter class:
   __init__(self, host: str = None, port: int = None, username: str = None,
             password: str = None, from_addr: str = None, use_mock: bool = False)
   - use_mock: check parameter AND os.getenv("USE_MOCK_SMTP", "false").lower() == "true"
   - If mock mode: store _use_mock=True, return early
   - If not mock:
     - host = host or os.getenv("SMTP_HOST", "smtp.office365.com")
     - port = int(port or os.getenv("SMTP_PORT", "587"))
     - username = username or os.getenv("SMTP_USERNAME")
     - password = password or os.getenv("SMTP_PASSWORD")
     - from_addr = from_addr or os.getenv("SMTP_FROM") or username
     - If username is None: raise ValueError("SMTP_USERNAME must be set or passed to SMTPAdapter")
     - If password is None: raise ValueError("SMTP_PASSWORD must be set or passed to SMTPAdapter")

2. send_alert(self, recipient: str, subject: str, body: str) -> None
   - If mock mode:
     Log at INFO: "[MOCK SMTP] Would send email to={recipient} subject='{subject}' body_preview='{body[:100]}'"
     Return without error.
   - If real:
     Build MIMEMultipart message (From, To, Subject, plain-text body)
     Open a new SMTP connection per send_alert:
       with smtplib.SMTP(host, port) as smtp:
           smtp.ehlo()
           smtp.starttls()
           smtp.login(username, password)
           smtp.send_message(msg)

3. Imports needed (all stdlib):
   import smtplib, os, logging
   from email.mime.multipart import MIMEMultipart
   from email.mime.text import MIMEText
   from typing import Optional

NOTE: A new SMTP connection per send_alert() is intentional — avoids stale connections
across long polling intervals. No persistent connection needed.
