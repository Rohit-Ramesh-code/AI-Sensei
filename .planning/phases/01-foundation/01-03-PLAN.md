  <action>
Implement adapters/ews_scraper.py. The EWSAdapter wraps exchangelib to send alert emails. It must support mock mode for development and real Exchange for production.

Structure:

1. Module-level: Read USE_MOCK environment variable pattern (same as SNMP adapter)

2. EWSAdapter class:
   __init__(self, server: str = None, username: str = None, password: str = None,
             sender_email: str = None, auth_type: str = None, use_mock: bool = False)
   - use_mock: check parameter AND os.getenv("USE_MOCK_EWS", "false").lower() == "true"
   - If mock mode: store use_mock=True, skip all exchangelib setup
   - If not mock:
     - server = server or os.getenv("EWS_SERVER")
     - username = username or os.getenv("EWS_USERNAME")
     - password = password or os.getenv("EWS_PASSWORD")
     - sender_email = sender_email or os.getenv("EWS_USERNAME")  (sender is the service account)
     - auth_type = auth_type or os.getenv("EWS_AUTH_TYPE", "NTLM")
     - If server is None: raise ValueError("EWS_SERVER must be set or passed to EWSAdapter")
     - Build exchangelib Account ONCE here (not in send_alert):
       credentials = Credentials(username=username, password=password)
       Map string auth_type to exchangelib constant:
         "NTLM" to NTLM, "BASIC" to BASIC, "MSAL" to raise NotImplementedError (Phase 1 scope)
       auth_map = {"NTLM": NTLM, "BASIC": BASIC}
       config = Configuration(server=server, credentials=credentials, auth_type=auth_map[auth_type.upper()])
       self._account = Account(primary_smtp_address=sender_email, config=config, autodiscover=False, access_type=DELEGATE)
     - If exchangelib raises during Account construction: re-raise with a clear prefixed message:
       "EWS connection failed (check EWS_SERVER, credentials, and auth type): {original error}"

   send_alert(self, recipient: str, subject: str, body: str) -> None
   - If mock mode:
     Log the email to Python logger (logging.getLogger(__name__)) at INFO level:
     "[MOCK EWS] Would send email to={recipient} subject='{subject}' body_preview='{body[:100]}'"
     Return without error.
   - If real:
     m = Message(
         account=self._account,
         subject=subject,
         body=body,
         to_recipients=[Mailbox(email_address=recipient)],
     )
     m.send()  (Does not save to Sent folder; use m.send_and_save() if copy is needed)

3. Imports needed:
   from exchangelib import Credentials, Configuration, Account, DELEGATE, Message, Mailbox, NTLM, BASIC
   import os, logging
   from typing import Optional

CRITICAL anti-patterns (per research):
- DO NOT build the Account in send_alert() — Account construction involves TLS handshake; build once at __init__
- DO NOT hardcode auth_type as NTLM — use EWS_AUTH_TYPE env var; IT may require BASIC or MSAL
